/** * A 'VPMCA' structure is what's stored in the ENOVIPropertyInfo object in the ENOVIObjectInfo. * For each attribute of each entity there's a VPMCA created when loading the package the entity * belongs to. VPMCA's are shared, there can only be one copy of a given attribute's VPMCA at any time * (this means that you must not refer to it by value, always by reference). VPMCA's are destroyed * when the PropertyInfo that owns them is destroyed, which happens when the PackageInfo (and thus * the ObjectInfo) is destroyed, that is at the end of the Enovia session. * **/ typedef struct AttributeDescription { void* _RealAttrDef; // SDAI attrdef. CATUnicodeString _Name; CATUnicodeString _Alias; CATUnicodeString _Magnitude; } VPMCA ; // #endif
