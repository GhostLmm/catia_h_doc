The subtype // contains the relevant data members used to print the report. // // Use this violation type to report violation on Objects in a Workpackage // while processing the workpackage in ValidateRootObject(). // The new violation type uses the ValidateRootObject and it only // needs to query the Workpackage to retrieve the information once and use them // for the rest of the products.
