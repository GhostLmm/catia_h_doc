To delete with the usual C++ <tt>delete</tt> operator after use. */ ExportedByCATTopologicalObjects CATPositionPtFaceOperator* CATCreatePositionPtFaceOperator(CATGeoFactory * iFactory, CATSoftwareConfiguration * iConfig, CATSurParam & iPoint, CATFace * iFace); /** * Creates an operator to test the position of a point with regards to a face. * @param iFactory * The factory of the geometry. * @param iConfig * The pointer to the Software Configuration * @param iNbPoints * The number of points to test in the array <tt>iPoints</tt>. * @param iPoints * The array <tt>[iNbPoints]</tt> of points to test, that must be on the surface of <tt>iFace</tt>. * @param iFace * The pointer to the face. * @return * The pointer to the created operator.
