Each color component ranges * from <code>0.</code> to <code>1.</code>. * @return * An HRESULT value. * <br><b> Legal values</b>: * <li><tt>S_OK</tt> the operation succeeded </li> * <li><tt>E_FAIL</tt> otherwise </li>. */ virtual HRESULT GetAmbient(double &oCoef, double oColor[3]) = 0; /** * Returns the diffuse parameters. * @param oCoef * The coefficient. * @param oColor * The RGB color.
